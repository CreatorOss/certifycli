🔐 CertifyCLI

Enterprise-grade Git commit signing made simple

No servers, no complexity - just secure, local-first signing

✨ Features

🚀 Local First

No servers required, works completely offline. Your keys never leave your machine.

🔐 Secure by Default

OS keychain storage, proper certificate management, and industry-standard cryptography.

⚡ One-Command Setup

Get started in under 60 seconds with our streamlined setup process.

🔍 Comprehensive Verification

Verify individual commits or entire repositories with detailed reporting.

🏁 Quick Start

# Setup your identity
certifycli setup

# Configure Git
certifycli git configure

# Start signing commits!
git add .
git commit -m "My securely signed commit"
